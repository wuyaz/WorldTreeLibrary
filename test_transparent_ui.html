透明背景设计特点：

  • 整体外壳透明: 移除了背景色和阴影，只保留边框
  • 头部半透明: 背景使用rgba(0, 0, 0, 0.1)轻微暗化
  • 按钮和输入框透明: 背景透明度从5%调整到3%
  • Edge标签页透明: 背景使用rgba(255, 255, 255, 0.03)
  • 聊天列表透明: 背景透明度从var(--SmartThemeBlurTintColor)调整为20%透明
  • 聊天卡片透明: 背景透明度从74%调整为30%
  • 批量操作栏透明: 背景从rgba(0,0,0,0.3)调整为rgba(0,0,0,0.15)
  • 置顶卡片透明: 渐变背景透明度降低，从78%/10%调整为30%/5%

这样的设计让聊天管理器更好地融入SillyTavern的背景，减少视觉干扰，同时保持清晰的边框和层次结构。
